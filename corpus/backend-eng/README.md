# Backend Eng Corpus

Role-specific knowledge for the Backend Eng agent. Stacks in play: FastAPI/Python (SIA) + Next.js/Node (richezamor.com). Scope: API endpoints, auth, DB layer, business logic, providers.

## Index

- [martin-kleppmann.md](./martin-kleppmann.md) — Author of *Designing Data-Intensive Applications*; the theoretical grounding for every data/consistency decision.
- [sebastian-ramirez-fastapi.md](./sebastian-ramirez-fastapi.md) — Creator of FastAPI, SQLModel, Typer; defines idiomatic Python API code for SIA.
- [mike-bayer-sqlalchemy.md](./mike-bayer-sqlalchemy.md) — Creator of SQLAlchemy and Alembic; authoritative on Python DB patterns, session lifecycle, migrations.
- [alex-xu.md](./alex-xu.md) — ByteByteGo / *System Design Interview* Vols 1–2; canonical patterns for APIs, rate limiting, caching, queues, sharding.
- [brandur-leach.md](./brandur-leach.md) — Ex-Stripe; the single best source on Postgres-first production patterns (idempotency keys, staged job drains, canonical log lines, API versioning).
- [owasp.md](./owasp.md) — Authoritative security reference: Top 10, API Top 10, ASVS, Cheat Sheet Series — non-optional for auth, input, and crypto decisions.
- [microsoft-google-api-guidelines.md](./microsoft-google-api-guidelines.md) — Microsoft Azure REST Guidelines + Google AIPs; the most mature public specs for API consistency, versioning, naming, pagination, errors.
