# Researcher corpus index

Role-specific knowledge for the Researcher agent. The Researcher owns Notion research pages, competitive matrices, and framework evaluations across three disciplines — competitive/market research, user-research synthesis, and technical framework evaluation — and presents options with tradeoffs rather than making product decisions.
