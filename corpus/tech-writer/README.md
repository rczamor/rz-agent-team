# Tech Writer Corpus

Role-specific knowledge for the Tech Writer agent on the 11-agent team. Scope: Notion spec hygiene, READMEs, docstrings, API docs, IDENTITY.md files. Docs-as-code fluency, API documentation standards, modern information architecture.

## Files

- [write-the-docs.md](./write-the-docs.md) — Write the Docs community: conferences, Slack, documentation principles, docs-as-code baseline.
- [daniele-procida-diataxis.md](./daniele-procida-diataxis.md) — Diátaxis framework: four-quadrant separation (tutorials, how-to, reference, explanation) for structuring any doc set.
- [google-dev-style-guide.md](./google-dev-style-guide.md) — Google Developer Documentation Style Guide: voice, terminology, API reference conventions; the default public style guide for dev docs.
- [microsoft-style-guide.md](./microsoft-style-guide.md) — Microsoft Writing Style Guide: warmer tone, strongest public reference on inclusive language and UI terminology.
- [stripe-twilio-docs.md](./stripe-twilio-docs.md) — Stripe and Twilio docs teams: the benchmark for API documentation — runnable code, interactive examples, narrative-to-reference flow.
- [tom-johnson-idratherbewriting.md](./tom-johnson-idratherbewriting.md) — Tom Johnson's I'd Rather Be Writing blog and free API documentation course; pragmatist voice, docs-as-code and AI-augmented workflows.
- [good-docs-gitlab-handbook.md](./good-docs-gitlab-handbook.md) — The Good Docs Project (open-source doc templates) and GitLab Handbook (public docs-as-source-of-truth at scale).
